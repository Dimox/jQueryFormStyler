# jQuery Form Styler <sup>[2.0.2](https://github.com/Dimox/jQueryFormStyler/blob/master/CHANGELOG.md)</sup>

jQuery-плагин для стилизации элементов HTML-форм:

- `<input type="checkbox">`
- `<input type="radio">`
- `<input type="file">`
- `<input type="number">`
- `<select>`
